# Settlement Prediction App

Developed by: Nathrah Hanim Nazri 🍀 

This app predicts 7-year embankment settlement using ANN, SVR, and GPR machine learning models.

The app allows users to:
- Enter embankment and soil parameters
- Predict settlement using ANN, SVR, and GPR
- Calculate average ML total settlement
- Recommend safe embankment height based on allowable settlement
